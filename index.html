3. Selectores CSS

Los selectores CSS son "patrones" que coinciden con elementos en el árbol HTML para aplicarles estilos. Aprende cómo apuntar a elementos específicos para darles vida.

Selector de ID: #miElementoUnico

Selecciona un elemento HTML único por su atributo id. Es el más específico y solo debe haber uno por página con el mismo ID.

Soy el elemento con ID: uniqueElement

Haz clic para aplicar/quitar estilo.

Selector de Etiqueta: p, div

Selecciona todos los elementos de un tipo de etiqueta HTML específico (ej., todos los párrafos <p>).

Selector de Clase: .miClase

Selecciona todos los elementos que comparten un atributo class específico. Permite reutilizar estilos en múltiples elementos.

Pseudoclases y Pseudoelementos

Pseudoclases (:hover, :focus): Estilizan elementos en un estado específico (ej., cuando el mouse está sobre ellos, o cuando un input está enfocado).

Pseudoelementos (::before, ::after): Crean y estilizan partes del elemento que no existen en el HTML (contenido generado).
