Introducción a CSS

CSS (Cascading Style Sheets) es un lenguaje de hojas de estilo que se utiliza para describir la presentación de un documento escrito en HTML. CSS describe cómo deben mostrarse los elementos en pantalla, en papel o en otros medios. Es fundamental para darle vida y estilo a nuestras páginas web.

En esta página, exploraremos tres conceptos poderosos que nos permiten crear experiencias de usuario dinámicas y visualmente atractivas: las Animaciones CSS, las Transiciones CSS y el elemento HTML <canvas> para dibujar con JavaScript.

a. Animaciones CSS

Las animaciones CSS nos permiten crear movimientos complejos y secuencias de estilo a lo largo del tiempo. A diferencia de las transiciones, las animaciones pueden tener múltiples "pasos" (keyframes) y se pueden controlar de forma más granular.

Conceptos Clave:

  • @keyframes: Define los diferentes estados de la animación en puntos específicos de tiempo (porcentajes).
  • Propiedades animation-*: Controlan cómo se aplica la animación a un elemento (duración, retardo, iteraciones, dirección, etc.).

Ejemplo Interactivo:

Selecciona una animación del menú desplegable y haz clic en "Iniciar Animación" para verla en acción.

Animado

Código de las Animaciones:


/* Animación 1: Cambio de Color y Movimiento */
@keyframes colorChangeAndMove {
    0% { background-color: #ef4444; transform: translateX(0); } /* Rojo */
    25% { background-color: #f97316; transform: translateX(50px); } /* Naranja */
    50% { background-color: #eab308; transform: translateX(0); } /* Amarillo */
    75% { background-color: #22c55e; transform: translateX(-50px); } /* Verde */
    100% { background-color: #0ea5e9; transform: translateX(0); } /* Azul */
}

/* Animación 2: Rotar y Escalar */
@keyframes rotateAndScale {
    0% { transform: rotate(0deg) scale(1); background-color: #facc15; } /* Amarillo */
    50% { transform: rotate(180deg) scale(1.2); background-color: #8b5cf6; } /* Violeta */
    100% { transform: rotate(360deg) scale(1); background-color: #facc15; } /* Amarillo */
}

/* Animación 3: Efecto de Pulso */
@keyframes pulseEffect {
    0% { transform: scale(1); box-shadow: 0 0 0 0 rgba(16, 185, 129, 0.7); } /* Esmeralda */
    70% { transform: scale(1.05); box-shadow: 0 0 0 10px rgba(16, 185, 129, 0); }
    100% { transform: scale(1); box-shadow: 0 0 0 0 rgba(16, 185, 129, 0); }
}

/* Aplicación de la animación al elemento (la animación específica se aplica con JS) */
.animated-box {
    animation-duration: 5s; /* Duración por defecto */
    animation-iteration-count: infinite; /* Repetición infinita */
    animation-direction: alternate; /* Alterna la dirección */
    animation-timing-function: ease-in-out; /* Curva de velocidad */
    animation-play-state: paused; /* Pausada por defecto */
}
.animated-box.running {
    animation-play-state: running;
}

b. Transiciones CSS

Las transiciones CSS nos permiten cambiar suavemente las propiedades de un elemento de un estado a otro en un período de tiempo determinado. Son perfectas para efectos simples como cambios de color, tamaño o posición al interactuar con un elemento (por ejemplo, al pasar el ratón por encima).

Conceptos Clave:

  • transition-property: La propiedad CSS a la que se aplicará la transición.
  • transition-duration: El tiempo que durará la transición.
  • transition-timing-function: La curva de velocidad de la transición (ej. ease, linear, ease-in-out).
  • transition-delay: Un retardo antes de que comience la transición.

Ejemplo Interactivo:

Pasa el ratón por encima del cuadro de abajo para ver la transición.

Transición

Código de la Transición:


.transition-box {
    background-color: #3b82f6; /* Azul */
    transition: background-color 0.5s ease-in-out,
                transform 0.5s ease-in-out,
                border-radius 0.5s ease-in-out;
}
.transition-box:hover {
    background-color: #ec4899; /* Rosa */
    transform: scale(1.1) rotate(5deg);
    border-radius: 50%; /* Se vuelve un círculo */
}

c. El Elemento HTML <canvas>

El elemento <canvas> de HTML5 proporciona una API de dibujo para renderizar gráficos 2D y 3D en el navegador utilizando JavaScript. No es CSS en sí mismo, pero es una herramienta poderosa para crear gráficos dinámicos que a menudo complementan los efectos visuales de CSS.

Conceptos Clave:

  • Es un "lienzo" en blanco: por sí mismo no dibuja nada.
  • Se necesita JavaScript para dibujar en él.
  • getContext('2d'): Obtiene el contexto de dibujo 2D para dibujar formas, texto, imágenes, etc.
  • beginPath(), moveTo(), lineTo(), stroke(), fill(): Métodos comunes para dibujar formas.

Ejemplo Interactivo:

Aquí hay un ejemplo simple de un círculo que se dibuja en el canvas y se mueve.

Código del Canvas:


<canvas id="myCanvas" width="400" height="200"></canvas>

<script>
    const canvas = document.getElementById('myCanvas');
    const ctx = canvas.getContext('2d');
    let x = 50;
    let dx = 2; // Velocidad del movimiento
    let animationFrameId;

    function drawCircle() {
        ctx.clearRect(0, 0, canvas.width, canvas.height); // Limpiar el canvas
        ctx.beginPath();
        ctx.arc(x, canvas.height / 2, 20, 0, Math.PI * 2); // Dibujar un círculo
        ctx.fillStyle = '#10b981'; // Color verde esmeralda
        ctx.fill();
        ctx.closePath();
    }

    function animateCanvas() {
        drawCircle();
        x += dx;
        if (x + 20 > canvas.width || x - 20 < 0) {
            dx = -dx; // Invertir dirección al tocar los bordes
        }
        animationFrameId = requestAnimationFrame(animateCanvas); // Bucle de animación
    }

    function startCanvasAnimation() {
        if (!animationFrameId) { // Evitar múltiples bucles
            animateCanvas();
        }
    }

    function stopCanvasAnimation() {
        cancelAnimationFrame(animationFrameId);
        animationFrameId = null;
    }

    // Dibujar el círculo inicial
    drawCircle();
</script>
